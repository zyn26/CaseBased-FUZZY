Install library yang dibutuhkan dengan perintah(untuk melihat dan membaca excel) : pip install pandas openpyxl

buka terminal, jalankan program dengan perintah py program.py (atau langsung dengan tombol run saja).

Program akan memproses data restoran menggunakan metode Fuzzy Logic.

Hasil output akan ditampilkan pada terminal dan otomatis disimpan dalam file: peringkat.xlsx

File tersebut berisi 5 restoran terbaik beserta nilai pelayanan, harga, dan skor kelayakannya.
